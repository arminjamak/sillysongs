# Silly Songs Website

## Setup Instructions

1. **Add the images:**
   - Save the "SILLY SONGS" rainbow logo as `silly-songs-logo.png`
   - Save the "Armin" signature as `armin-signature.png`

2. **Open the website:**
   - Simply open `index.html` in your web browser
   - Or use a local server for better performance

3. **Features:**
   - Rotatable 3D dinosaur model (click and drag to rotate, scroll to zoom)
   - Audio player with play/pause and scrubbing functionality
   - Fully responsive design
   - Beautiful gradient audio player

## Files:
- `index.html` - Main HTML structure
- `style.css` - Styling and responsive design
- `script.js` - 3D model viewer and audio player logic
- `dinosaur.glb` - 3D dinosaur model
- `song.wav` - "Attracted to Narcissists" audio file
- `silly-songs-logo.png` - Top logo (you need to add this)
- `armin-signature.png` - Bottom right signature (you need to add this)
